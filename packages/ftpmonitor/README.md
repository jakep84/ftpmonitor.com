# ftpmonitor (CLI)

Diagnose FTP / FTPS / SFTP endpoints instantly from your terminal.

Runs real protocol checks:

**DNS → TCP → Auth → List**

## Install

```bash
npm i -g ftpmonitor
```

## Usage

### SFTP (password)

```bash
ftpmonitor check --protocol sftp --host example.com --username user --password pass --path .
```

### SFTP (private key)

```bash
ftpmonitor check --protocol sftp --host example.com --username user --privateKey ~/.ssh/id_rsa --path .
```

### FTP

```bash
ftpmonitor check --protocol ftp --host 127.0.0.1 --port 2121 --username test --password test
```

### Output formats (for tickets)

```bash
ftpmonitor check --protocol sftp --host example.com --username user --password pass --format slack --copy
ftpmonitor check --protocol sftp --host example.com --username user --password pass --format jira
ftpmonitor check --protocol sftp --host example.com --username user --password pass --format markdown
```

### Open troubleshooting link

```bash
ftpmonitor check --protocol sftp --host example.com --username user --password pass --open
```

## Remote mode (optional)

Use the hosted API instead of running the checks locally:

```bash
ftpmonitor check --api https://ftpmonitor.com/api/health-check --protocol sftp --host example.com --username user --password pass
```

## Exit codes

- `0` success
- `2` check failed (one or more steps failed)
- `3` usage / validation error
