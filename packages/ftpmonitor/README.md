# ftpmonitor

Diagnose FTP / FTPS / SFTP endpoints instantly from your terminal.

Runs real protocol checks:

DNS → TCP → Auth → Directory List

---

## 🚀 Try It Instantly (No Install Required)

```bash
npx ftpmonitor check --protocol sftp --host example.com --username user --format slack --copy
```

### JSON output (CI friendly)

```bash
npx ftpmonitor check --protocol sftp --host example.com --username user --format json
```

---

## 📦 Install Globally

```bash
npm install -g ftpmonitor
```

Then:

```bash
ftpmonitor check --protocol sftp --host example.com --username user --password pass
```

---

## 🔎 What It Checks

For any FTP / FTPS / SFTP endpoint:

1. DNS Resolution
2. TCP Connectivity
3. Authentication
4. Directory Listing

Each step includes timing, status, and structured output.

## 🔐 SFTP with Private Key

```bash
ftpmonitor check   --protocol sftp   --host example.com   --username user   --privateKey ~/.ssh/id_rsa   --passphrase yourpassphrase
```

---

## 🌐 Optional: Use Hosted API

Run checks from ftpmonitor.com instead of locally:

```bash
ftpmonitor check   --api https://ftpmonitor.com/api/health-check   --protocol sftp   --host example.com   --username user   --password pass
```

---

## 🧠 Exit Codes

0 → Success  
2 → One or more checks failed  
3 → Usage / validation error

## Why ftpmonitor Exists

FTP-based systems are still heavily used in:

- Healthcare data exchange
- Financial batch processing
- Government integrations
- Vendor file drops
- Legacy B2B workflows

When they fail, diagnosing the issue is often slow and manual.

ftpmonitor provides a fast, structured way to validate endpoints without configuring a local client.
